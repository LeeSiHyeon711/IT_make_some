# 실행 안내 — PickUpMemo v2

> 이 문서는 PickUpMemo v2 앱을 스마트폰에 설치하고 처음 사용하는 방법을 단계별로 안내합니다.
> 개발 지식이 없어도 그대로 따라 하실 수 있습니다.

---

## 앱 소개 (한 줄 요약)

배민커넥트 배차 화면에 등록해 둔 업체가 나타나면, 화면 위에 내가 적어 둔 메모를 자동으로 팝업으로 띄워 주는 Android 앱입니다.

---

## 준비물

- Android 스마트폰 (Android 8.0 이상)
- APK 파일: `app-debug.apk` (아래 경로에서 PC로 찾을 수 있습니다)
  - PC 경로: `projects/PickUpMemo_v2/05-개발/PickUpMemo_v2/app/build/outputs/apk/debug/app-debug.apk`
  - 파일 크기: 약 5.8 MB
- 배민커넥트(`com.woowahan.bros`) 앱이 스마트폰에 설치되어 있어야 합니다

---

## 1단계. 스마트폰에 APK 파일 옮기기

APK 파일을 PC에서 스마트폰으로 옮겨야 합니다. 편한 방법 중 하나를 선택하세요.

**방법 A — USB 케이블로 연결**
1. USB 케이블로 PC와 스마트폰을 연결합니다.
2. 스마트폰에 "파일 전송" 또는 "MTP" 모드를 선택합니다.
3. PC 탐색기에서 스마트폰 저장소가 열리면 `app-debug.apk` 파일을 스마트폰의 "다운로드" 폴더에 복사합니다.

**방법 B — 구글 드라이브 등 클라우드 이용**
1. PC에서 `app-debug.apk` 파일을 구글 드라이브(또는 카카오톡 파일 전송 등)에 업로드합니다.
2. 스마트폰에서 해당 파일을 다운로드합니다.

---

## 2단계. 앱 설치하기

1. 스마트폰의 **파일 관리자**(또는 "내 파일") 앱을 엽니다.
2. "다운로드" 폴더로 이동하여 `app-debug.apk` 파일을 탭합니다.
3. **"알 수 없는 앱 설치 허용"** 안내가 나오면 "허용"을 선택합니다.
   - 스마트폰 제조사마다 표현이 조금 다를 수 있습니다. "이 출처의 앱 허용" 또는 "허용하고 설치"를 선택하면 됩니다.
4. 설치가 완료되면 "열기"를 탭하거나, 앱 서랍에서 **"PickUpMemo"** 아이콘(보라색 배경에 흰 말풍선 심볼)을 찾아 실행합니다.

---

## 3단계. 필수 권한 설정 (반드시 두 가지 모두 허용해야 합니다)

앱이 배민커넥트 화면을 감지하고 팝업을 띄우려면 아래 두 가지 권한이 모두 필요합니다. 하나라도 빠지면 팝업이 동작하지 않습니다.

앱을 실행하면 메인 화면에 현재 권한 상태가 표시됩니다.

### 권한 1 — 접근성 서비스 켜기

이 권한은 배민커넥트 화면에 어떤 텍스트가 표시되는지 앱이 읽을 수 있게 해 줍니다.

1. 앱 메인 화면에서 **[접근성 설정]** 버튼을 탭합니다.
2. 시스템 접근성 설정 화면이 열립니다. 목록에서 **"PickUpMemo"** 항목을 찾아 탭합니다.
   - "설치된 앱" 또는 "다운로드한 앱" 섹션에 있습니다.
3. 스위치를 **"켬(ON)"** 으로 전환합니다.
4. 시스템 경고 대화상자가 나타나면 **"허용"** 을 탭합니다.
5. 뒤로 가기 버튼으로 PickUpMemo 앱으로 돌아옵니다.
6. 메인 화면에서 "접근성 서비스: **실행 중**" 으로 바뀐 것을 확인합니다.

### 권한 2 — 다른 앱 위에 표시 허용

이 권한은 팝업이 배민커넥트 화면 위에 떠 있을 수 있게 해 줍니다.

1. 앱 메인 화면에서 **[오버레이 권한 설정]** 버튼을 탭합니다.
2. "다른 앱 위에 표시" 설정 화면이 열립니다. 자동으로 PickUpMemo 항목이 표시됩니다.
3. 스위치를 **"허용"** 으로 전환합니다.
4. 뒤로 가기 버튼으로 PickUpMemo 앱으로 돌아옵니다.
5. 메인 화면에서 "오버레이 권한: **허가됨**" 으로 바뀐 것을 확인합니다.

> 두 항목 모두 정상으로 표시되어야 팝업이 동작합니다.

---

## 4단계. 메모 등록하기

팝업에 표시할 업체 메모를 앱에 미리 등록합니다.

1. 메인 화면에서 **[메모 관리]** 버튼을 탭합니다.
2. 오른쪽 하단의 **[+] 버튼**(동그란 플로팅 버튼)을 탭합니다.
3. 아래 항목을 입력합니다.

   | 항목 | 설명 | 필수 여부 |
   |------|------|-----------|
   | 상호명 | 업체 이름 (예: 푸라닭) | 필수 |
   | 지점명 | 지점 이름 (예: 신림점) | 필수 |
   | 메모내용 | 기억해 둘 내용 (예: 지하 주차장, 2층 계단 이용) | 필수 |
   | 태그 | 분류용 단어 (예: 주차) | 선택 |

4. 입력 후 **[저장]** 버튼을 탭합니다.

> 상호명과 지점명을 모두 입력해야 정확한 매칭이 됩니다. 예를 들어 "푸라닭"과 "신림점"을 따로 입력해야 "푸라닭 신림점"만 팝업이 뜨고, "푸라닭 봉천점"에는 뜨지 않습니다.

---

## 5단계. 팝업이 제대로 동작하는지 미리 확인하기 (권장)

실제 배차를 받기 전에 앱 안에서 팝업을 미리 테스트해 볼 수 있습니다.

1. 메인 화면에서 **[팝업 테스트]** 버튼을 탭합니다.
2. **모드 A (간단 확인)**: 상단 목록에서 등록한 메모를 선택하고 **[선택한 메모로 팝업 띄우기]** 버튼을 탭합니다. 화면 상단에 카드 형태의 팝업이 나타나는지 확인합니다.
3. 팝업이 약 6초 후 자동으로 사라지는지 확인합니다.

> 팝업이 나타나지 않는다면 3단계 권한 설정이 모두 완료되었는지 다시 확인하세요.

---

## 6단계. 실제 사용하기

1. PickUpMemo 앱을 백그라운드로 내립니다. (홈 버튼 또는 홈 제스처)
2. 배민커넥트 앱을 실행하여 배차 대기 상태로 둡니다.
3. 신규 배차 카드가 화면에 나타날 때, 등록한 업체가 포함된 배차라면 화면 상단에 자동으로 팝업이 표시됩니다.
4. 팝업에는 업체 제목(상호명 + 지점명)과 등록한 메모 내용이 표시됩니다.
5. 팝업은 약 **6초 후 자동으로 사라집니다.** 별도로 닫을 필요가 없습니다.

---

## 배민 로그 조회 및 내보내기 (선택 기능)

앱이 배민커넥트 화면에서 수집한 접근성 로그를 확인하고 파일로 내보낼 수 있습니다.

1. 메인 화면에서 **[배민 로그]** 버튼을 탭합니다.
2. 수집된 로그 목록과 건수가 표시됩니다.
3. **[내보내기]** 버튼을 탭하면 txt 파일로 내보내기(공유) 화면이 열립니다.
4. **[전체 삭제]** 버튼을 탭하면 확인 후 로그를 모두 삭제할 수 있습니다.

---

## 자주 묻는 점과 주의 사항

**Q. 재부팅 후 팝업이 안 떠요.**
접근성 서비스는 일부 기기에서 재부팅 후 자동으로 꺼질 수 있습니다. PickUpMemo 앱 메인 화면에서 "접근성 서비스: 실행 중"인지 확인하고, 꺼져 있으면 [접근성 설정] 버튼으로 다시 켜 주세요.

**Q. 같은 배차 카드에서 팝업이 한 번 뜬 뒤 30초 동안 다시 안 떠요.**
정상 동작입니다. 동일한 메모에 대해 30초 안에 팝업이 중복으로 뜨는 것을 막는 기능입니다.

**Q. 배민커넥트가 아닌 다른 앱(카카오맵, 카카오톡 등)에서 팝업이 뜨나요?**
뜨지 않습니다. 배민커넥트(`com.woowahan.bros`) 앱에서 발생한 이벤트만 감지합니다.

**Q. 앱을 강제 종료하면 어떻게 되나요?**
접근성 서비스가 함께 종료될 수 있습니다. 앱 메인 화면에서 권한 상태를 다시 확인하고, 필요하면 설정에서 재활성화해 주세요.

**Q. 상호명만 입력하고 지점명을 비워도 되나요?**
안 됩니다. 상호명과 지점명 두 가지가 모두 배차 카드 텍스트에 포함될 때만 팝업이 표시됩니다. 지점명을 비우면 저장이 되지 않습니다.

**Q. 태그는 꼭 입력해야 하나요?**
태그는 선택 항목입니다. 비워 두면 팝업에 태그 영역이 표시되지 않습니다.
