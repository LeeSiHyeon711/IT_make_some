# 고객 전달문 — PickUpMemo v2

안녕하세요. IT상상공방입니다.
요청하신 PickUpMemo v2가 완성되어 전달드립니다.

---

## 무엇을 만들었나요

배민커넥트 배차 화면에서 등록해 두신 업체가 감지되면, 별도 앱 전환 없이 화면 위에 메모가 자동으로 팝업되는 Android 앱을 만들었습니다.

앱을 백그라운드에 두고 배민커넥트를 켜 두기만 하면, 신규 배차 카드에 등록한 상호명과 지점명이 모두 포함된 경우 6초간 카드 형태 팝업으로 메모를 보여 줍니다. 서버, OCR, 인터넷 연결 없이 스마트폰 안에서만 동작합니다.

---

## 이번 MVP에 포함된 기능

| 기능 | 설명 |
|------|------|
| 메모 관리 (CRUD) | 업체별 메모 등록·조회·수정·삭제. 상호명·지점명·메모내용(필수) + 태그(선택) |
| 자동 팝업 | 배민커넥트 신규 배차 카드에서 등록 업체 감지 시 화면 상단에 6초간 오버레이 팝업 표시 |
| 중복 억제 | 동일 메모에 대해 30초 이내 팝업 중복 표시 없음 |
| 노이즈 차단 | 배민커넥트 이외 앱(카카오맵·카카오톡·시스템UI 등)으로 인한 오탐 없음 |
| 팝업 검증 테스트 화면 | 실제 배차 없이 앱 안에서 팝업 동작을 직접 확인할 수 있는 테스트 화면 |
| 권한 상태 안내 | 메인 화면에서 접근성 서비스·오버레이 권한 현재 상태 표시 및 설정 이동 버튼 제공 |
| 배민 로그 조회·내보내기 | 앱이 수집한 배민커넥트 접근성 로그를 목록으로 확인하고 txt 파일로 내보내기 |
| 적응형 앱 아이콘 | 보라색 배경 + 흰 말풍선 심볼로 런처에서 식별 가능 |

## 확인 포인트 (직접 확인해 주세요)

아래 항목은 실제 기기에서 직접 확인이 필요합니다.

1. 설치 후 접근성 서비스와 오버레이 권한을 모두 허용했는지 확인하세요. (메인 화면에서 두 상태가 모두 "실행 중" / "허가됨"이어야 합니다.)
2. 메모를 1건 이상 등록한 뒤 [팝업 테스트] 화면에서 팝업이 정상 표시되는지 먼저 확인하세요.
3. 실제 배민커넥트에서 등록한 업체의 배차 카드가 들어올 때 팝업이 화면 상단에 나타나는지 확인하세요.

## MVP 범위에서 제외한 항목 (향후 확장 후보)

아래 기능들은 이번 MVP에 포함되지 않았습니다. 필요하시면 다음 단계로 요청해 주세요.

**1. 배차 수락 후 상세 화면 감지**
배민커넥트에서 배차를 수락한 뒤 나타나는 상세 화면(가게전화 패턴)에서도 업체명을 감지하는 기능입니다. v1에서 수집 가능함이 확인되었으나, 이번 MVP는 신규 배차 카드 시점만 대상으로 했습니다.

**2. 다른 배달 플랫폼 확장**
쿠팡이츠, 요기요 등 다른 배달 앱 화면에서도 메모를 감지하는 기능입니다.

**3. 메모 백업·복원**
등록한 메모를 파일로 내보내거나 다른 기기에 옮길 수 있는 기능입니다. 현재는 앱을 삭제하면 메모도 삭제됩니다.

**4. 태그 검색·필터**
등록된 메모 목록에서 태그로 검색하거나 분류별로 필터하는 기능입니다.

**5. 팝업 표시 시간 조정**
현재 6초로 고정된 팝업 표시 시간을 사용자가 직접 설정할 수 있는 기능입니다.
