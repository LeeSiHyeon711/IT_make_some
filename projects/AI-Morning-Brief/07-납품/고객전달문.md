## 만들어드린 것

**AI 업계 뉴스를 매일 아침 자동으로 분석해 Discord에 전달하는 파이프라인**입니다.

사람이 개입하지 않아도 매일 새벽 4시 30분에 Mac이 혼자 깨어나 AI 관련 뉴스 사이트 7곳을 돌며 기사를 수집하고, Claude가 각 기사를 분석해 "왜 중요한지", "IT상상공방에 어떻게 활용할 수 있는지", "오늘 실험해볼 만한 것이 무엇인지"를 정리해 Discord 채널에 전송합니다. 운영자는 아침에 Discord를 열면 이미 분석된 브리핑이 와 있는 상태를 경험하게 됩니다.

## 운영 전 채워야 할 값 3가지

아래 세 가지를 직접 설정해야 파이프라인이 실제로 동작합니다. 구체적인 방법은 동봉된 실행안내.md를 참고하세요.

### 1. Anthropic API 키 (Claude 분석용)

`.env` 파일의 `ANTHROPIC_API_KEY` 항목에 입력합니다.

```
ANTHROPIC_API_KEY=sk-ant-여기에_실제_키_붙여넣기
```

- 발급처: https://console.anthropic.com
- 없으면: Claude 분석 대신 키워드 기반 기본 요약으로 자동 전환됩니다. 리포트는 생성됩니다.

### 2. Discord Webhook URL (브리핑 수신용)

`.env` 파일의 `DISCORD_WEBHOOK_URL` 항목에 입력합니다.

```
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/여기에_URL
```

- 발급 방법: Discord 앱 > 채널 설정(톱니바퀴) > 연동 > 웹훅 > 새 웹훅 > URL 복사
- 없으면: Discord 전송을 건너뛰고 로컬 파일만 저장됩니다. 자동화 자체는 동작합니다.

### 3. RSS 소스 목록 (수집 대상 뉴스 사이트)

`config/sources.yaml` 파일을 열어 원하는 RSS 피드 목록을 확인·수정합니다.

```
cp config/sources.example.yaml config/sources.yaml
```

예시 파일에 7개 소스가 들어 있으며 `enabled: false`로 비활성화하거나 원하는 소스를 추가할 수 있습니다. 실제 접속이 되는지는 `python3 main.py --check-sources` 명령으로 확인하세요.
