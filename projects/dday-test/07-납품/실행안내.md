# 실행 안내 — dday-test

이 문서를 따라 하면 D-Day 카운터를 직접 확인하고 사용하실 수 있습니다.

---

## 무엇인가요

날짜를 고르면 오늘부터 그 날까지 며칠이 남았는지(또는 며칠이 지났는지)를 즉시 표시해 주는 D-Day 카운터 웹 페이지입니다. 별도 설치나 인터넷 연결 없이, 파일을 더블클릭하는 것만으로 바로 쓸 수 있습니다.

---

## 확인하는 법 (순서대로)

### 1단계. 파일 위치 찾기

전달받은 폴더를 열면 아래 구조로 파일이 들어 있습니다.

```
projects/
  dday-test/
    05-개발/
      index.html    ← 이 파일을 엽니다
      style.css
      script.js
```

`05-개발` 폴더 안에 있는 **`index.html`** 파일을 찾습니다.

### 2단계. index.html 파일 열기

`index.html` 파일을 **더블클릭**합니다.

- Chrome, Safari, Edge, Firefox 등 평소 쓰는 인터넷 브라우저 창이 뜨면서 D-Day 카운터 화면이 나타납니다.
- "날짜를 선택하세요" 라는 안내 문구가 표시되면 정상입니다.

> 파일을 더블클릭해도 아무 일이 일어나지 않으면, 파일에 마우스 오른쪽 버튼을 클릭하고 "다음으로 열기" → 브라우저 이름(Chrome 등)을 선택하세요.

### 3단계. 날짜 입력하기

화면 중앙에 입력 영역이 두 개 나타납니다.

1. **(선택) 이벤트 이름 입력**: "이벤트 이름" 칸에 기억하기 쉬운 이름을 적습니다. (예: 토익시험, 여름휴가, 제출 마감)  입력하지 않아도 괜찮습니다.
2. **목표 날짜 선택**: "목표 날짜" 칸을 클릭하면 달력이 나타납니다. 원하는 날짜를 고릅니다.

날짜를 고르는 즉시(버튼 클릭 없이) 결과가 화면 하단에 나타납니다.

---

## 무엇을 보게 되나요

날짜를 선택하면 화면 아래쪽에 결과가 크고 굵게 표시됩니다.

| 상황 | 화면에 표시되는 내용 (예시) |
|------|---------------------------|
| 오늘로부터 21일 뒤, 이벤트명 "토익시험" | **토익시험 D-21** |
| 오늘이 바로 그 날 | **D-0** |
| 이미 4일이 지난 날짜 | **D+4** |
| 날짜를 아직 선택하지 않은 상태 | **날짜를 선택하세요** |

---

## 사용 시나리오 3가지 예시

### 시나리오 1. 앞으로 남은 날 확인 (미래 날짜)

> 다음 달 시험이 며칠 남았는지 확인하고 싶을 때

1. 이벤트 이름에 "토익시험"을 입력합니다.
2. 목표 날짜에서 시험 날을 고릅니다.
3. 화면에 **"토익시험 D-21"** 처럼 남은 일수가 즉시 표시됩니다.

### 시나리오 2. 오늘이 바로 그 날 (당일)

> 오늘이 행사 당일인지 확인하고 싶을 때

1. 목표 날짜에 오늘 날짜를 고릅니다.
2. 화면에 **"D-0"** 이 표시됩니다.

### 시나리오 3. 지나간 날짜 확인 (과거 날짜)

> 어떤 날이 며칠 전이었는지 돌아보고 싶을 때

1. 목표 날짜에 과거 날짜를 고릅니다.
2. 화면에 **"D+4"** 처럼 지난 일수가 표시됩니다.

## 알려진 한계 (이런 것은 되지 않습니다)

| 되지 않는 것 | 이유 |
|-------------|------|
| 페이지를 새로고침하면 입력한 내용이 사라집니다 | 저장 기능이 이번 버전에는 포함되지 않았습니다 |
| D-Day를 여러 개 동시에 관리하는 목록이 없습니다 | 한 번에 하나의 날짜만 계산하도록 만들어졌습니다 |
| D-Day 임박 알림이 울리지 않습니다 | 알림 기능은 이번 범위에 포함되지 않았습니다 |
| 스마트폰 앱이 아닙니다 | 웹 페이지이므로 브라우저에서만 사용 가능합니다 |

## 잘 안 될 때

**화면이 아예 열리지 않을 때**
- `index.html` 파일이 맞는지 확인하세요. `style.css`, `script.js`와 같은 폴더에 함께 있어야 합니다.
- 파일 하나만 다른 곳으로 옮기면 디자인이나 계산이 동작하지 않습니다. 세 파일을 항상 같은 폴더에 보관하세요.

**결과가 표시되지 않을 때**
- 날짜를 선택한 뒤에도 결과가 보이지 않으면 브라우저를 닫고 `index.html`을 다시 더블클릭해 보세요.
- 인터넷 익스플로러(IE)는 지원하지 않습니다. Chrome, Safari, Edge, Firefox를 사용하세요.

**날짜 계산이 이상한 것 같을 때**
- 이 페이지는 사용 중인 기기의 날짜와 시간을 기준으로 계산합니다. 기기의 날짜 설정이 올바른지 확인해 보세요.
