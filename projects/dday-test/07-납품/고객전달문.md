# D-Day 카운터 MVP를 전달드립니다

안녕하세요. 요청 주신 D-Day 카운터를 만들어 전달드립니다.

---

## 만든 것

날짜를 고르면 오늘을 기준으로 남은 일수(D-21처럼) 또는 지나간 일수(D+4처럼)를 즉시 화면에 표시해 주는 웹 페이지입니다. 이벤트 이름(예: "토익시험")을 함께 입력하면 "토익시험 D-21"처럼 이름과 함께 보여 줍니다.

설치나 회원가입이 전혀 필요 없고, 인터넷이 없어도 됩니다. 전달드린 파일에서 `index.html`을 더블클릭하는 것만으로 바로 열립니다.

---

## 확인 부탁드립니다

직접 보시는 방법은 함께 첨부된 **실행안내.md** 를 참고해 주세요. (파일 열기 → 날짜 선택 → 결과 확인, 3단계면 됩니다)

특히 이 부분이 의도와 맞는지 봐주시면 감사하겠습니다.

- 날짜를 고르는 순간 버튼 없이 즉시 결과가 바뀌는 느낌이 편하신가요?
- 이벤트 이름을 함께 표시하는 방식("토익시험 D-21")이 원하시던 형태인가요?

---

## 이번에 포함한 것

| 기능 | 내용 |
|------|------|
| 날짜 선택 | 달력 UI로 연·월·일을 고릅니다 |
| D-Day 즉시 계산 | 버튼 없이 날짜를 고르는 즉시 계산됩니다 |
| 미래·당일·과거 표시 | D-21 / D-0 / D+4 형식으로 각각 구분해 표시됩니다 |
| 이벤트 이름 표시 | 이름 입력 시 "이벤트명 D-n" 형태로 함께 표시됩니다 (선택 사항) |
| 모바일 지원 | 스마트폰 화면에서도 잘 보이도록 맞춰져 있습니다 |

---

## 현재 되지 않는 것 (알고 계시면 좋은 것)

- **저장이 안 됩니다**: 브라우저를 새로고침하거나 페이지를 닫으면 입력한 날짜가 사라집니다. 다시 들어오시면 새로 선택해 주셔야 합니다.
- **D-Day를 여러 개 동시에 볼 수 없습니다**: 한 번에 하나의 날짜만 계산됩니다.
- **알림이 없습니다**: D-Day가 다가왔을 때 자동으로 알려주는 기능은 없습니다.

이번 MVP는 "일단 빠르게 확인해 보는" 목적으로 최소한의 기능만 담았습니다. 사용해 보시고 불편하거나 더 필요한 것이 있으면 바로 말씀해 주세요.
