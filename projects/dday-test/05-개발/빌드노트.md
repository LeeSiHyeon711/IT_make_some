## #4 formatLabel 검증 근거
Node로 함수 단위 점검 결과 (라벨 형식 + 이벤트명 접두):

- 당일 + 이벤트명: (0, "토익시험") → **"토익시험 D-0"**
- 당일 + 이벤트명 없음: (0, "") → **"D-0"**
- 과거 + 이벤트명: (-4, "토익시험") → **"토익시험 D+4"**
- 과거 + 이벤트명 없음: (-4, "") → **"D+4"**
- 공백 이벤트명 정리: (-4, "  여행  ") → **"여행 D+4"** (trim 적용)
