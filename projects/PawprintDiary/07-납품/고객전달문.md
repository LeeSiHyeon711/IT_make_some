# 고객 전달문 — 발자국일기 (PawprintDiary)

안녕하세요.
IT상상공방입니다.

요청해 주신 **발자국일기**의 1차 MVP가 완성되었습니다.
무엇을 만들었는지, 어떻게 사용하시면 좋을지, 앞으로 어떻게 발전시킬 수 있는지를 정리해 드립니다.

---

## 무엇을 만들었나요

강아지·고양이 보호자가 하루를 짧게 기록하면,
AI가 그 기록을 읽고 컨디션 요약·행동 해석·관찰 포인트·추억 문장을 따뜻하게 정리해 주는
**AI 반려동물 일기 웹앱**입니다.

회원 가입 없이 바로 시작할 수 있고, 기록은 사용하시는 브라우저 안에 안전하게 저장됩니다.
AI 키를 별도로 넣지 않아도 전체 기능을 체험할 수 있습니다.

---

## 핵심 기능 다섯 가지

### 1. 반려동물 프로필 등록
이름, 종, 나이, 성별, 성격, 좋아하는 것, 싫어하는 것, 건강 특이사항, 프로필 사진을 등록합니다.
AI가 답변할 때 이 정보를 참고해 우리 아이에게 맞춤화된 응답을 만들어 줍니다.

### 2. 오늘의 발자국 기록
식욕, 활동량, 수면, 배변을 클릭 한 번으로 선택하고, 기분 태그와 특이 행동, 자유 일기, 사진까지 남길 수 있습니다.
바쁜 하루에도 **1~2분이면 충분**하도록 만들었습니다.

### 3. AI 발자국 읽기
일기를 저장하면 AI가 자동으로 네 가지를 정리해 줍니다.

- **오늘의 컨디션**: 기록을 바탕으로 오늘 몸 상태를 부드럽게 요약
- **행동 이야기**: 특이 행동이나 기분의 가능한 의미를 탐색
- **내일의 관찰 포인트**: 다음 날 살펴보면 좋을 부분 안내
- **오늘의 한 줄 추억**: 오늘 하루를 기억할 따뜻한 문장

식욕이나 활동량이 평소보다 많이 떨어진 날에는 "가까운 시일 내 수의사 선생님과 상담해 보시길 부드럽게 권해드려요" 같은 안내 카드가 추가됩니다.
AI는 의료 진단을 하지 않고, 반려동물의 말을 대신하지 않으며, 보호자에게 죄책감을 주는 표현을 쓰지 않습니다.

### 4. AI에게 질문하기
"오늘 밥을 잘 안 먹는데 왜 그럴까요?", "요즘 유난히 졸려 보이는데 괜찮은 걸까요?" 같은 질문을 자유롭게 입력하면
AI가 등록된 프로필과 최근 기록을 참고해 맞춤 답변을 드립니다.
대화 내역은 저장되어 나중에 다시 확인할 수 있습니다.

### 5. 기록 모아보기
지난 기록을 날짜별로 목록으로 보고, 항목을 클릭하면 당시 입력 내용과 AI 요약을 함께 확인할 수 있습니다.
기록이 쌓일수록 우리 아이의 패턴과 소중한 순간이 한 곳에 모입니다.

---

## 사용 시나리오 — 하루 2분, 이렇게 쓰세요

1. **저녁, 퇴근 후** → 앱을 열고 "오늘의 발자국" 탭 클릭
2. 식욕·활동량·수면·배변을 각각 클릭 한 번으로 선택
3. 오늘의 기분 태그("활발해 보임", "차분함" 등)를 고르고
4. 특이한 점이 있다면 짧게 메모 ("산책 중 낯선 개를 보고 짖었어")
5. "발자국 남기기" 버튼을 누르면 AI가 자동으로 오늘의 기록을 정리해 줍니다
6. 궁금한 점이 생기면 "질문" 탭에서 바로 물어보면 됩니다

## MVP 범위와 이번에 포함되지 않은 것

이번 1차 버전에서 만든 것과, 처음부터 다음 단계로 미룬 것을 명확히 안내드립니다.

### 이번 버전에 포함된 것
- 반려동물 프로필 1마리 등록
- 오늘의 발자국 기록 (식욕/활동량/수면/배변/기분/사진/자유일기)
- AI 4종 요약 + 병원 상담 권장
- AI 질문하기 (기록 기반 맞춤 답변)
- 기록 목록 및 상세 조회
- 기록 삭제
- 모든 데이터 브라우저 로컬 저장 (회원 가입 불필요)

### 처음부터 이번 버전에서 제외한 것
| 항목 | 이유 |
|------|------|
| 회원 가입 / 로그인 | MVP 핵심 흐름 검증 우선, 2차에서 추가 |
| 다기기 동기화 | 로컬 저장 기반 MVP, 2차에서 서버 저장 연동 |
| 다중 반려동물 관리 | 1마리 흐름 먼저 검증, 화면 구조는 확장 대비 설계 완료 |
| 커뮤니티·SNS 공유 | MVP 범위 외 |
| 병원 예약 연동 | MVP 범위 외 |
| 모바일 앱 패키징 (Android/iOS) | MVP 범위 외, 웹에서 먼저 검증 |
| 이별 후 회고 모드 | MVP 범위 외 |
| 주간/월간 리포트 | MVP 범위 외 |

## 사소하지만 솔직하게 알려드리는 것 (비블로킹 관찰 2건)

검수 중 기능 문제는 아니지만 사용 경험과 관련해 기록해 두는 내용입니다.

**관찰 1. "오늘의 발자국" 탭에 이미 기록이 있어도 새 폼이 표시됨**
오늘 이미 기록을 작성한 후 탭을 다녀와서 다시 들어가면, 기존 기록 대신 새 입력 폼이 표시됩니다.
기록 자체는 "기록" 탭에서 언제든지 다시 확인할 수 있으며 사라지지 않습니다.
하루에 여러 번 기록을 남기는 것도 가능하도록 설계되어 있습니다.
다음 버전에서는 당일 기록이 있으면 먼저 보여주는 방식으로 개선할 수 있습니다.

**관찰 2. 브라우저 탭 아이콘(파비콘)이 없음**
브라우저 탭에 발자국일기 전용 아이콘이 표시되지 않습니다. 기능에는 전혀 영향이 없습니다.
다음 버전에서 발자국 모양 아이콘을 추가할 수 있습니다.

## 다음 단계 제안 (2차 확장 후보)

MVP를 사용해 보신 후 아래 중 원하시는 방향으로 발전시킬 수 있습니다.

| 우선순위 (제안) | 기능 | 한 줄 설명 |
|----------------|------|-----------|
| 높음 | 다기기 동기화 + 회원 인증 | 다른 기기·가족과 기록을 공유, 백업 |
| 높음 | 다중 반려동물 관리 | 두 마리 이상의 기록 분리 관리 |
| 중간 | 주간/월간 컨디션 리포트 | AI가 한 주/한 달 변화를 정리해 알려줌 |
| 중간 | 컨디션 변화 감지 알림 | 평소보다 떨어진 날 자동 안내 |
| 중간 | 추억 문장 모아보기 | 그동안의 한 줄 추억만 타임라인으로 |
| 낮음 | 사진첩 | 기록한 사진만 모아 보는 화면 |
| 낮음 | 이별 후 회고 모드 | 함께한 시간을 소중히 돌아보는 별도 공간 |
| 낮음 | 모바일 앱 패키징 | 홈 화면 설치, 알림 기능 |

수정 요청이나 새 기능 추가는 언제든지 말씀해 주시면 순서대로 진행하겠습니다.
