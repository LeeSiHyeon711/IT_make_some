# prompts/ — 에이전트 프롬프트 (예정)

이 폴더에는 향후 **상상공방 Lite 에이전트들의 실제 프롬프트**를 보관할 예정입니다.

구현 단계에서 다음 에이전트별 프롬프트가 들어옵니다 (기획: [`../docs/`](../docs/)):

- 소통 에이전트 (자유 서술 → 구조화 + 최소 질문 + 시간 예산 수집)
- 아이디어 진단 에이전트 (균열점 1개 진단, `pain_source`별 분기)
- 첫 실험 설계 에이전트 (시간 예산 기반 첫 검증 미션)
- 출력 에이전트 (공방 결과카드 편집)

> 아직 비어 있습니다. 각 프롬프트는 해당 기획 문서(`docs/03`~`docs/07`)의 역할·원칙·입출력 JSON을 기준으로 작성합니다.
