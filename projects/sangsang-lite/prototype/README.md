# prototype/ — 프로토타입·데모 구현 (예정)

이 폴더에는 향후 **상상공방 Lite의 PlayMCP/MCP 프로토타입 또는 데모 구현**을 보관할 예정입니다.

예상 산출물 (기획: [`../docs/`](../docs/)):

- 카카오 PlayMCP 연동 프로토타입 (또는 MCP 서버/도구 형태)
- 6단계 순차 워크플로 데모 (자유 서술 → 접수 확인 → 균열점 진단 → 첫 검증 미션 → 공방 결과카드)
- 데모 시나리오([`../docs/09-demo-scenarios.md`](../docs/09-demo-scenarios.md)) 실행 예시

> 아직 비어 있습니다. MVP는 **순차 실행**만 구현하며, 병렬 실행/오케스트레이션은 후속 고도화 단계입니다.
