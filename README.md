# IT상상공방 (IT Make Some)

> 고객의 상상을 **빠르게 확인 가능한 MVP**로 만들어주는 AI 자동 생산 공방.

## 한눈에 보기

고객의 막연한 아이디어가 들어오면, 8종의 전담 AI 에이전트가 작업해
**웹 / 앱 / 자동화 프로세스** 형태의 MVP로 만들어 "이게 맞나요?"를 즉시 검증한다.

```
   ── 게이트(사람 승인) ──        ── 자동 ──        ── 게이트 ──
1상담 ▣ 2기획 ▣ 3설계 ▣ ═▶ 4이슈등록 5개발 6자동QA ─▶ ▣사람 수동테스트 ─▶ 7납품 → 고객
                       설계승인=개발착수            증상 시 ↻ 분석→이슈→자동수정→재QA
```

- **휴먼 게이트는 두 곳뿐**: ①방향 결정(상담·기획·설계) ②실사용 검증(사람 수동 테스트). 그 사이는 자동.
- 4단계 GitHub 관리자가 설계를 이슈로 박제해 중간에 끊겨도 흐름을 잃지 않는다.
- 수동 테스트에서 나온 증상은 증상분석가(triage)가 재현·분석해 자동 수정 순환으로 처리한다.
